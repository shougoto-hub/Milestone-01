How to push in GitHub

  • git init
  • git add . (all files)
  • git commit -m "comment"
  • git branch -M main
  • git remote add origin https://github.com/shougoto-hub/practice.git (remote origin copy from repository)
  • git push -u origin main

After 1st time - then:

  • git add . (all files)
  • git commit -m "comment"
  • git push
